El posicionamiento float toma dos valores posibles, “left” (izquierdo) o “right” (derecho), cuando tenemos este tipo de posicionamiento, todos los elementos con este valor de atributo flotan en la dirección indicada, es decir que si tiene posicionamiento "float:left" comenzaran a ordenarse desde la izquierda uno tras otro mientras le quede espacio en la pantalla y si no seguirán abajo. Al variar el tamaño de la pantalla se reacomodan automáticamente. Para que se dejen de comportar de esta manera se utiliza el atributo “clear” que puede tomar los valores “left", "right" o "both".
